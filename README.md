# CCNA 200-301

> Story-based CCNA 200-301 notes and hands-on Packet Tracer labs based on **Jeremy's IT Lab**.

## 📖 About

This repository contains my personal study notes and Packet Tracer labs while preparing for the **Cisco Certified Network Associate (CCNA 200-301)** certification.

The notes are designed to be **concise, beginner-friendly, and easy to review**. Rather than copying the lecture transcript, each lesson focuses on defining key concepts with short explanations and simple examples.

The labs reinforce the theory through hands-on practice using **Cisco Packet Tracer**.

> **Note:** These notes are based on Jeremy's IT Lab CCNA course and are intended to complement the video lessons, not replace them.
